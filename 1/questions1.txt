1. С++ - компилируемый(собирает .exe файл), мультипарадигменный (поддерживает несколько парадигм программирования:
 1.Функциональное 2.ООП 3.Обобщенное), сочетает низко(позволяет работать с памятью) и высокоуровневые средства,
 со строгой статической типизацией (тип переменной сохраняется во время работы программы).
2. а)boolean(bool) - логический тип данных, занимает 1 байт, false=0, true любое другое число
б)character(char) - символьный тип данных, занимает 1 байт, каждому символу сопоставлено число от 0 до 255 по таблице ASCII
в)integer(short, int, long) - целочисленный тип данных, занимает 2,4,8 байт
г)floating point(float,double) - сожердит число с плавующей точкой 4,8 байт 
д)void - пустой тип данных
3. Проблема переносимости возникает когда работающаю на одной системе программа, перерстает работать коректно
 на другой. Это может произойти при различных таблицах ASII(следует использовать только символы соответствующие
 числам от 0 до 127), если в системе отводится другое количество памяти под какой-то из типов данных(следует писать
 код, не основывающийся на количестве памяти, отводимой под какой-либо тип данных)
4. а) Объявление - обозначаем только имя и тип данных переменной
б) Определение - к предыдщему добавляется вся необходимая информация, например выделенная память
в) Инициализация - первое присваивание значения переменной
г) Присваивание - последующие присваивания значения переменной
5. Выравнивание данных упрощает процессору доступа к данным в оперативной памяти, чтобы при обращении к памяти ему
требовалось меньше операций. Например, процессор, считвающий по 4 байта за раз, может считать требуемый
нам объем информации с 1 по 4 байт в памяти за одну операцию доступа . Однако считывание данных, ннапример, с 2 по 5 байты в два раза увеличивает количество операций доступа к памяти, требуемых для считывания того же
объема данных. Поэтому важно в правильном порядке задавать перемнные, чтобы они были выравниванены по памяти.
